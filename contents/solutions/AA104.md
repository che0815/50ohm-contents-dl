Die Symbolrate wird in $\unit{\baud}$ (Baud) angegeben.   

$\qty{1}{\baud} = \qty{1}{\text{Symbol}\per\second}$  

Ein Symbol kann aus einem oder mehreren bits bestehen.

Ausschlussverfahren: Bit ist eine Informationseinheit, Hertz ist eine Frequenz und Dezibel ist eine logarithmische Hilfsmaßeinheit zur Darstellung von Pegeln oder Verhältnissen.
